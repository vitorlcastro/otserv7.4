QUICK START:
============

1. Clone repository:
   git clone <url> tibia-server
   cd tibia-server

2. Build and start:
   docker-compose build
   docker-compose up -d

3. Initialize database:
   ./scripts/init-database.sh

4. Verify:
   docker-compose ps

5. Connect:
   Use Tibia 7.4 client to connect to localhost:7171
   Test account: testaccount / testpassword


PRODUCTION DEPLOYMENT:
======================

Run the automated deployment script:
./scripts/deploy-production.sh

This will:
- Build Docker images
- Start all services
- Initialize database
- Configure firewall
- Set up monitoring
- Generate deployment report
